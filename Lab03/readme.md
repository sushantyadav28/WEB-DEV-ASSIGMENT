#Lab Assignment-03 ---  Personal Portfolio Website Using Advanced CSS 

**Assignment Title:**
One-Page Responsive Portfolio Website using Flexbox, CSS Grid & Animations.

**Project Overview:**
This project is a one-page responsive portfolio website built using *Flexbox, **CSS Grid, **responsive design and **animations.

**Learning Outcomes:**

**By completing this assignment, I learned:**
1. How to use Flexbox to align and position elements.
2. How to design layouts using CSS Grid.
3. Creating responsive layouts using media queries.
4. Applying hover effects, transitions, and simple animations.
5. Structuring a complete one-page modern portfolio website.
6. Improving UI using spacing, colors, and typography.
