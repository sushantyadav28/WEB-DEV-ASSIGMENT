**JavaScript Console Quiz Game**

A simple and interactive JavaScript Quiz Game that runs entirely inside the browser console.
The game asks multiple quiz questions using prompt(), checks the user's answers, gives immediate feedback using alert(), and displays the final score at the end.

**Learning Outcomes**

1. By completing this project, you will learn to:
2. Use arrays to store multiple pieces of data
3. Apply for loops to iterate over a list
4. Write reusable functions
5. Take user input through prompt()
6. Display messages using alert()
7. Use string manipulation methods for validation
8. Implement conditional logic (if-else)
